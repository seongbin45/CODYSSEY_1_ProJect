# 확인봇 입력 템플릿 (재사용)

아래 블록을 복사해 채운 뒤 확인봇에 붙여넣으세요.

```
[업무 과업] 정책/절차 확인 이메일 작성
[배경 상황]
- (승인 이력, 금액, 업체명, 관련 문서 상태 등)

[확실히 모르는 것]
1)
2)
3)

[타겟] 담당 매니저 (정중·사무적 실무 관계)
[톤] 정중하고 간결한 실무 메일체
[분량] 본문 250~350자
[금지] 이미 배경에 명시된 내용 재질문 금지 / 예·아니오로 끝나는 순서 질문 금지
[서명] (예: 김철수 드림)
```

## 최소 입력 체크리스트

- [ ] 배경에 **확정 사실**이 1개 이상 있는가?
- [ ] **모르는 것**이 항목으로 나열되어 있는가?
- [ ] 이미 답이 있는 내용을 다시 묻지 않았는가?
- [ ] 순서 질문이 있으면 "지금 무엇부터?"까지 포함했는가?
