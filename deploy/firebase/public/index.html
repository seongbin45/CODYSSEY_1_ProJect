보너스 1 · 재사용 배포 패키지

✉️ 확인봇 프롬프트 패키지

창업지원 실무 이메일 작성 코치 — 시스템 프롬프트, Few-shot, 입력 템플릿을 파일 단위로 공유합니다. Firebase Hosting 등으로 정적 배포할 때 쓰는 랜딩 페이지입니다.

다운로드 / 원본 파일

  • system_prompt.md — 시스템 프롬프트 v2
  • input_template.md — 입력 템플릿
  • few_shot.json — Few-shot 3건
  • package_index.json — 패키지 인덱스 (재사용 메타데이터)

어떻게 쓰나요?

  • Custom GPT: Instructions에 system_prompt.md 내용을 붙여넣기
  • Streamlit: 저장소에서 streamlit run bot/app.py
  • 직접 사용: 입력 템플릿을 채운 뒤 아무 LLM에 시스템 프롬프트와 함께 전달

핵심 규칙 (한 줄 요약)

  • 모호하면 되묻기 · 없는 수치/정책 단정 금지
  • 순서 질문은 “지금 무엇부터?”까지
  • 확인사항 최대 3개 · 톤/채널 변경 시 내용 유지
